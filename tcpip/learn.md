# 关键字

  当前层报头，有效载荷

# 四层协议，由顶层开始
  1. Application layer(应用层)

      Telnet,FTP,e-mail

  2. Transport layer(传输层)

      TCP and UDP

  3. Network layer(网络层)

      IP,ICMP,IGMP

  4. physical layer(链路层)
      
      设备驱动程序，这是最底层。协议包括了ARP(Address Resolution Protocol)

# 数据单元
  1. TCP segment
  2. IP datagram
  3. Frame(链路层称“帧”)
