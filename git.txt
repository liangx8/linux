git rev-parse <tag>
常用命令
git config --global --edit
git config set --global core.editor nano

缺省配置路径 $HOME/.gitconfig

git与服务器连接时，会使用ssh作为中间件，所以当github连不上时，是因为ssh连不上。
此时需要为ssh连接代理(注意:不是为git设置代理)
  用到netcat命令，需要用openbsd-netcat。编辑ssh设置文件 ~/.ssh/config,添加以下
  内容：
        Host github.com
            ProxyCommand nc -X 5 -x 127.0.0.1:1080 %h %p
